Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll()

Question-2: what are the different between innerHTML, innerText and textContent?

innerHTML returns the HTML content of an element, including tags and spacing, while innerText returns the rendered text content (excluding HTML tags and hidden elements), and textContent returns all text content, including hidden text and script/style elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript that allows you to attach a single event listener to a parent element to handle events for all of its child elements, even if those children are added dynamically

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM event propagation mechanism where an event triggered on a child element "bubbles up" through its parent hierarchy.
